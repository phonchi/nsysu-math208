Topological Sort — DFS 後序反轉得到拓撲順序

›按「開始」執行 DFS-based 拓撲排序
速度
Recursion stack： empty
result（尚無結束的頂點）
topo（DFS 結束後反轉 result）
虛擬碼 CODE
def topsort(graph): result = [] visited = set() def dfs(v): visited.add(v) for u in v.neighbors(): if u not in visited: dfs(u) result.append(v) # 後序：離開時加入 for v in graph: if v not in visited: dfs(v) return reversed(result) # 反轉 → 拓撲順序
用途
拓撲排序適用於 DAG（有向無環圖）：給每條邊 u→v 一個線性順序，使 u 永遠出現在 v 之前。常見於課程先修、編譯依賴、build 系統等。
複雜度
時間O(|V| + |E|)
空間O(|V|)
