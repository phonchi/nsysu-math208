def _perc_up(self, i):     while (i - 1) // 2 >= 0:         p = (i - 1) // 2         if heap[i] < heap[p]:             swap(heap[i], heap[p])         i = p
用途
Insert：把新 key 放到陣列尾端（保結構），再 perc_up 與 parent 比較交換直到滿足 heap 性質，總成本 $O(\log n)$。
