BST 插入 (put) — 沿 BST 性質找空位

›輸入 key 後按 put 進行插入

虛擬碼 — _put CODE
def _put(key, val, cur):     if key < cur.key:         if cur.left:             _put(key, val, cur.left)         else:             cur.left = TreeNode(key, val)     else:         if cur.right:             _put(key, val, cur.right)         else:             cur.right = TreeNode(key, val)
