BST 搜尋 (get) — 沿樹下移比較 key

›輸入 key 後按 get 進行搜尋
速度
虛擬碼 — _get CODE
get(self, key, currentNode):     if ! currentNode: return NULL     if currentNode->key == key:         return currentNode     else if key < currentNode->key:         return get(key, currentNode->leftChild)     else:         return get(key, currentNode->rightChild)
